Project Background

Course: ROB 456 - Cognitive Robotics, Oregon State University, Spring 2026

Team Members: Zachary Aszmus, Lucas Dassonville, Perry Johnston, Benjamin Molnar, and Quang Huy Nguyen

Project Focus: The goal of the project is to create two robots that can play a capture the flag style game, where there are four randomly placed ArUco tag “goals”. Each robot has two goals and scores points by placing the correct color barrels by their associated tags.

Why This Project?

We are doing this because we wanted to challenge ourselves with the course content we learned and implemented over the course of the term. Having four separate goal positions as well as 4 separate goal objectives would keep the project unique.

Final Project

Project Description

The project creates two robots that can play a capture the flag style game, where there are four ArUco tag goals. Each robot considers two ArUco tags to be a goal, one for each color barrel, which are orange and blue. Points are scored when a robot picks up and places a barrel at the correct ArUco tag. The robots have the ability to steal from the opposing robot, where it pilots to an opponent's goal, takes a barrel, and brings it to the goal for that color barrel. Success is based on barrel placement success rate and closeness to the correct ArUco tag.
